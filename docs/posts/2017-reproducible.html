Reproducible results with set.seed

example

This post demonstrates that random simulation outputs can be made reproducible by using set.seed().

Author

Kyle Baron

Published

01/01/2017

  • 2 Single dose
  • 3 Multi dose

library(mrgsolve)
library(dplyr)

1 Demo model

code <- '
$PKMODEL cmt="CENT"
$PARAM CL = 1, V = 20

$OMEGA 1 1

$TABLE
capture ETA1 = ETA(1);
capture ETA2 = ETA(2);
'
mod <- mcode("test", code)

2 Single dose

set.seed(9911)
single <- mod %>% ev(amt=100) %>% mrgsim(nid=100, end=24)

3 Multi dose

set.seed(9911)
multi <- mod %>% ev(amt=100, ii=24, addl=9) %>% mrgsim(nid=100, end=240)
